# selfoss-karfa-data

Sjálfvirk gagnasöfnun fyrir Selfoss Körfubolti.

Þetta repository keyrir Python scraper á klukkutíma fresti í gegnum GitHub Actions og vistar leikjagögn úr KKÍ mótakerfi í `data/games.json`.

## Gögn

Gögn eru aðgengileg á:
```
https://bjorgvinrunar.github.io/selfoss-karfa-data/data/games.json
```

## Keyrsla

GitHub Actions keyrir sjálfkrafa á klukkutíma fresti. Hægt er að keyra handvirkt undir **Actions → Sækja KKÍ leikjagögn → Run workflow**.
